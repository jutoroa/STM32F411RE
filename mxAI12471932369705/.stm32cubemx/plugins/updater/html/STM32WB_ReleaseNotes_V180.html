STM32CubeWB Firmware Package V1.8.0 / 23-June-2020
Main Changes

Add Zigbee low power mode support and firmware and wireless stack upgrade over the air
  • Zigbee
    • Add Low power mode support.
    • Add OTA support.
    • Add Persistent data storage support.
    • Add APS layer access.
    • Some Zigbee application are now provided on 3 IDEs.
    • New application notes available.

    • AN5506: Getting started with ZigBee on STM32WB
    • AN5498: How to use Zigbee clusters templates on STM32WB Series
    • AN5500: ZSDK API implementation for ZigBee on STM32WB Series
    • AN5491: Creating Manufacture Zigbee Specific Clusters on STM32WB Series
    • AN5492: ZigBee persistent data management non-volatile memory for STM32WB Serie

  • BLE
    • BLE Light wireless stack
    • Add support for Slave mode only, Privacy mode, Secure connection and Whitelist.

  • Replace SW4STM32 by STM32CubeIDE projects files for all examples and application.
