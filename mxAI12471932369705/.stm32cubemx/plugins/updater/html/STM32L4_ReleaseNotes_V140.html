STM32CubeL4 Firmware Package V1.4.0 / 26-February-2016
Main Changes
  • Official release of HAL, Low Layer drivers and Middlewares to add support of STM32L431xx/STM32L432xx/STM32L433xx/STM32L442xx/STM32L443xx devices.
  • 68 projects for Examples, Applications (FreeRTOS, USB Device) and Demonstration for 7 segment Gravitech display shield for STM32L432KC-Nucleo board.
  • USB device applications for STM32L476G-Discovery board.
