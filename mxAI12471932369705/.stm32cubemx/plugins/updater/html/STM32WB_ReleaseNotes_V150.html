STM32CubeWB Firmware Package V1.5.0 / 14-February-2020
Main Changes

Introduction of STM32WB5Mxx, STM32WB35xx, STM32WB30xx product and BLE/Zigbee static concurrent mode
  • Add support of STM32WB35xx, STM32WB30xx and STM32WB5Mxx
  • Introduction of STM32CubeIDE usage inside the examples projects
  • MAC 802.15.4: Add low power and External PA support
  • Zigbee: Add the support of several new cluster
  • Add BLE/Zigbee static concurrent mode support
