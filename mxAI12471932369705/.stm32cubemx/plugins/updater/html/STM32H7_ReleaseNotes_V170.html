STM32CubeH7 Firmware Package V1.7.0 / 14-February-2020
Main Changes
  • Maintenance release to support Value Line on STM32H7B3I-EVAL.
  • Add support of OSPI multiplexed mode feature.
  • Add source code for STM32H7B3I-EVAL demonstrations.
  • Add support of Keil and STM32CubeIDE projects for STM32H7B3I-DK demonstrations.
  • Add source code for STM32H743I-EVAL demonstrations.
